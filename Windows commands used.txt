【Deploy 22H2 Update with Microsoft Update StandalonePackage (MSU) Manually】
 
Step 1: Download the .MSU file from where it is instructed
 
1. Move the downloaded file to C:\KB5041582 *create this folder & head
to C:\temp and create another folder named ‘5041582’ *Named after
the KB.
ーーーーーーーーーーーー
Step 2: Run CMD as administrator and extract file
1. Run the script:
Expand –F:* c:\KB5041582\windows10.0-kb5041582-x64_aef566aad27209be83d504162b09f910f43ae2a5.msu
 c:\temp\5041582
 
2. The .cab files are extracted to c:\temp\5041582
 
3. On the same command prompt run the following script, 
DISM.exe /Online /Add-Package /PackagePath:c:\temp\5041582\windows10.0-kb5041582-x64.cab
 
4. Head to Windows Update for the 22H2 Update
ーーーーーーーーーーーー
Step 3. If windows update is stuck or not appearing
1. net stop wuauserv = stops the windows update service
2. net start wuauserv = starts the service (if not restarted
automatically)
3.Continue running the update *this may take several tries
ーーーーーーーーーーーー
Step 4. Confirm 22H2 Version
1. Head to Entra look for the non-complaint device.
2. The version should start with .19045 (22H2)
