Used in Nano to do a quick test a website is up or down.

1. create a nano file
nano for_test_ping_domains.sh

2. run the script
bash for_test_ping_domains.sh

=====================
【Script used】

#!/bin/bash

for x in google.com bing.com;
do
  if ping -q -c 2 -W 1 $x > /dev/null; then
    echo "$x is up"
  else
    echo "$x is down"
  fi

done
