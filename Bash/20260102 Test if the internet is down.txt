Used in Nano to do a quick test to see if the internet is up or down.

1. Create a nano file
nano for_test_internet_down.sh

2. Run the script
bash test_internet_down.sh

=====================
【Script used】

#!/bin/bash

#Is my internet down?

echo "Which IP address  do you wan to check?"
read target

while true
do
  if  ping -q -c 2 -W 1 $target > /dev/null; then
     echo "You're up!"
     break
  else
     echo "$target is currently down."
  fi
Sleep 2

done
